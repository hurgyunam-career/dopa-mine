# 현재 구현 상태
- `app_flutter`에서 `UC-01` 흐름이 동작한다: 홈에서 5개 운동 종목 선택, `운동 시작` 버튼 활성화, 세션 화면 이동.
- `UC-02` 반영으로 세션 화면에서 시간(타이머)과 횟수(수동 카운트)를 함께 수집한다.
- `운동 완료` 시 세션 데이터(운동 ID, 시작 시각, 소요 시간, 횟수, 완료 여부)를 저장하고 리포트 화면으로 이동한다.
- 저장 성공 이후 리포트 진입 시 포인트 지급 알림(SnackBar)을 즉시 노출한다.
