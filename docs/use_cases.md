# Use Cases: Dopa-Mine

## UC-01: 즉시 운동 진입 (Main Flow)
- **Goal**: 사용자가 고민 없이 운동을 시작하게 함.
- **Actor**: User
- **Steps**:
  1. 홈 화면에서 5개 종목(Bird-dog 등) 중 하나를 탭함.
  2. 하단 '운동 시작' 버튼이 활성화됨.
  3. 버튼 클릭 시 해당 종목의 측정 화면으로 이동.

## UC-02: 데이터 수집 및 세션 완료
- **Goal**: 운동 수행 데이터를 무결하게 저장.
- **Steps**:
  1. 수동 또는 센서로 데이터(시간/횟수) 수집.
  2. '운동 완료' 클릭 시 세션 데이터를 서버(Supabase)로 전송.
  3. 성공 시 포인트 지급 알림 및 리포트 화면 노출.
